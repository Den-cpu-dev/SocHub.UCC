# SocHub.UCC - Department of Sociology & Anthropology Website

A comprehensive department website for the Department of Sociology & Anthropology at the University of Cape Coast.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dark Mode Support**: Toggle between light and dark themes with localStorage persistence
- **Accessible**: WCAG 2.1 compliant with keyboard navigation and ARIA labels
- **Complete Course Catalog**: All Sociology (SOC 101-499) and Anthropology (ANT 101-499) courses with descriptions
- **Faculty Directory**: 20+ faculty members with contact information and specializations
- **Interactive Modals**: Click on courses to view detailed information including career paths
- **SEO Optimized**: Open Graph tags, JSON-LD structured data, meta descriptions
- **Course Materials**: Google Drive integration for Level 100-400 course materials
- **Scholarship Database**: Links to 12+ international scholarship opportunities

- **HTML5**: Semantic markup with proper accessibility

- Skip-to-main-content link

- Mobile menu with proper semantics

- Chrome/Edge (latest 2 versions)

## How to Use

1. Open `index.html` in a web browser
2. Navigate using the main menu or anchor links
3. Click on any course to view details
4. Click on faculty cards to view profiles
5. Toggle dark mode using the button in the header
6. Use the hamburger menu on mobile devices
